Excel Template Format

Your Excel file must contain the following columns:

  • Hotel Code - Unique hotel identifier (required)
  • Hotel Name - Full hotel name (required)
  • City - Hotel location city (required)
  • Inventory - Number of rooms (required)
  • Hotel Link - Website URL (optional)
  • Address Link - Google Maps link (optional)
Tip: If a hotel with the same Hotel Code already exists, it will be updated with the new information.
